## [1.0.0] - 2026-06-27

### Features
- Affichage de la météo en temps réel (API Open-Meteo)
- Gestion d'état avec Provider + MVVM
- Support de 4 villes : Cotonou, Parakou, Lagos, Abidjan
- Photo de ville personnalisable (galerie + caméra)
- Localisation GPS pour trouver la ville proche
- Notifications d'alerte chaleur (T > 33°C)
- Tests unitaires complets (codes WMO, conditions dangereuses)
- Animations fluides (fondu, Hero, rotation soleil)
- Couleur de fond dynamique selon température
- Affichage détaillé de la météo avec prévisions

### Technical
- Flutter 3.22.0
- Dart 3.11.5
- Provider pour la gestion d'état
- Dio pour les requêtes API
- Geolocator pour GPS
- Flutter Local Notifications
- Image Picker pour les photos
